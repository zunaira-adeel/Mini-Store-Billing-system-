# Mini-Store-Billing-system-
The objective of this lab assignment is to develop a C program that simulates a basic store  billing system. The program should take input for multiple items, calculate totals, apply  discount and tax, and display a formatted bill.


## Objective

The objective of this lab assignment is to develop a C program that simulates a basic store billing system. The program takes input for multiple items, calculates totals, applies discounts and tax, and displays a formatted bill.

---

# Introduction

In this project, we developed a mini store billing system using the C programming language. The program allows the user to enter item prices and quantities, then calculates the total bill amount automatically. It also applies discount according to specified conditions and adds tax to determine the final payable amount.

The billing system demonstrates the practical implementation of programming concepts such as arrays, loops, pointers, conditional statements, and menu-driven operations. It provides a simple real-world application of C programming for beginners.

---

# Program Outline

1. Take input for number of items
2. Store item prices and quantities using arrays
3. Calculate total cost for each item
4. Compute grand total
5. Apply discount based on total amount
6. Add tax to the bill
7. Display formatted bill details
8. Find the most expensive item
9. Use menu-driven operations for different tasks
